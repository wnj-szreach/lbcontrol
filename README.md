block_lbcontrol

插件类型：block
Moodle版本:2.0+
媒体中心版本:1.1.28.6GNT.2_MOODLE.1+

版本记录

a)用于Moodle和媒体中心融合，从Moodle上面直接实时监控媒体中心各个教室。
